# Personal Homepage

This is a static academic homepage for GitHub Pages.

## Files

- `index.html`: page content and structure
- `styles.css`: visual design and responsive layout
- `script.js`: publication search and filters

## Deploy To GitHub Pages

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Upload these files to the repository root.
3. Go to `Settings -> Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select `main` and `/root`.
6. Visit `https://<your-github-username>.github.io/` after GitHub finishes publishing.

## Customize

Replace placeholder content in `index.html` and update the `publications` array in
`script.js`.
